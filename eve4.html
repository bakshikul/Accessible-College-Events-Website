Accessibility Testing with Screen Readers

April 25, 2026


Overview


This event provides a live demonstration of testing websites using popular screen readers to ensure real-world accessibility compliance.


Topics Covered


  • Introduction to screen readers.
  • NVDA (Windows) and VoiceOver (macOS) demos.
  • How screen readers interpret HTML.
  • esting forms, buttons, and navigation.
  • Common accessibility issues detected by screen readers

Who Should Attend


  • Frontend developers.
  • Accessibility advocates.
  • UX designers and testers.

Outcome


Participants will learn how to test their websites using assistive technologies and improve accessibility for visually impaired users.
