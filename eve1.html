Web Accessibility Workshop

March 15, 2026


Overview


This workshop introduces the core principles of web accessibility and inclusive design.
Participants will learn how to build websites that are usable by people with diverse abilities, including users of assistive technologies.


Topics Covered


  • Introduction to WCAG 2.1 guidelines.
  • Color contrast and readable typography.
  • Semantic HTML and accessible structure.
  • Keyboard navigation and focus management.
  • Common accessibility mistakes and how to fix them.

Who Should Attend


  • Web developers and designers.
  • Beginners interested in accessible development.
  • Students working on real-world web projects.

Outcome


Participants will gain practical knowledge and hands-on experience to start building accessible websites confidently.
